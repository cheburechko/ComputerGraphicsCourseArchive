Программа для рисования полилиний и заливок.

Алгоритм Брезенхема - drawers.PolylineDrawer.draw
span-алгоритм - drawers.FillDrawer.draw

Полилинии

Для начала рисования полилинии нужно нажать левую кнопку мыши. 
После этого программа будет отрисовывать линию от последней поставленной точки
до указателя мыши, если он находится внутри поля рисования. Для установки
каждой следующей точки нужно нажимать левую кнопку мыши. Для завершения
рисования нужно нажать правую кнопку мыши (по нажатию новая точка не ставится).

Рисование полилиний выбрано по умолчанию, ширина по умолчанию - 1.

Заливка

Для рисования заливки нужно просто кликнуть ЛКМ в нужной точке.
Цвет по умолчанию - красный , связность - 4.

Первые 4 кнопки на тулбаре = меню File: новый файл, открыть, сохранить, выйти.
Следующие 2 кнопки = меню Polyline:  выбрать рисование полилиний, переключить толщину между 1 и 3.
Следующие 5 кнопок = меню Fill: выбрать рисование заливки, красный/синий цвет, связность 8/4.
Последняя кнопка = меню Help: About.

Реализация

Пакет models содержит классы модели: абстрактный класс Drawable, наследник Observable, который только
умеет возвращать свой тип в виде enum DrawableType, а также Canvas, Fill, Polyline.
Canvas - коллекция Drawable. При добавлении объекта передает новый объект в notifyObservers 
для перерисовки.

Пакет drawers содержит для каждого класса модели класс, отрисовыающий ее в BufferedImage.
Здесь реализованы все алгоритмы рисования.

Пакет serializers содержит классы для перевода модели в текстовое представление и обратно (read/write).
Абстрактный класс DrawableSerializer имеет функцию для пропуска комментариев skip, остальные 
классы наследуются от него.

Класс View совмещает в себе графическое представление и контроллер.

Класс Main - главное окно. 

При ошибках чтения из файла программа выдает ошибку с текстом исключения.
