Программа для применения фильтров

Все фильтры лежат в пакете FF_11312_Cherenkov_Filt.filter
Матричные - в FF_11312_Cherenkov_Filt.filter.matrix, абстрактный класс MatrixFilter
Пиксельные в FF_11312_Cherenkov_Filt.filter.pixel, абстрактный класс PixelFilter
Акврелизация - отдельно, WatercolorFilter

Для отображения изображений используется ImageClip.

По умолчанию загружается файл defalut.png. 

В настройках матричного преобразования делитель должен быть положительным числом,
порог от 0 до 255. Делитель считается по кнопке "Apply divisor". 
Если сумма чисел в матрице равна 0, делитель будет 1.

Кнопки для копирования и фильтры расположены на тулбаре.
